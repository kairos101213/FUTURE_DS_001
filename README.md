FUTURE_DS_01 – Ad Performance Analytics Dashboard

Internship Program
Future Interns – Data Science & Analytics Internship

Task Description
This task focuses on analyzing digital advertising performance data
(from Facebook/Instagram Ads) to evaluate campaign success and provide
actionable business insights.

Objectives
- Track key advertising KPIs such as Impressions, Clicks, CTR, CPC, and ROI
- Identify top-performing ads and campaigns
- Analyze trends across audience segments
- Provide recommendations to improve future campaigns

Tools Used
- Power BI (Primary dashboarding tool)
- Microsoft Excel (Data cleaning and preparation)

Repository Structure
- data/raw/ – Original ad performance dataset
- data/processed/ – Cleaned dataset used in Power BI
- visuals/ – Dashboard screenshots and charts
- report/ – Business insights and recommendations
- .pbix file – Interactive Power BI dashboard

Deliverable
An interactive Power BI dashboard with filters, KPIs, and insights
to support marketing decision-making.
